### How Web Page Loads
- Parse / Load the HTML
- Fetch external resources like CSS (Blocks the rendering), JS (Blocks the parsing)
- Using CSS creating CSSOM
- Execute JS
- Merge DOM and CSSOM, then creates render tree
- Calculate and Paint
