# Accessibility Standards

This has 3 standard levels
- Level A - Baisc
- Level A - Intermediate
- Level AAA - High

### WEBAIM / WCAG Principles
- Perceivable - Anyone on web they should consume
- Operable - Anyone on web they should can do some operations
- Understandable - Anyone on web they should should understand
- Robust - Should work accross multiple various devices
